doctree モジュールのディレクトリ構造について

```
refm            リファレンスマニュアル
refm/spec               Ruby言語仕様
refm/api                クラスリファレンス
refm/capi               C API リファレンス
refm/news               リリースノート
faq             FAQ
faq/tk                  Ruby/Tk FAQ
faq/trap                旧「Rubyの落とし穴」
```

* 旧「Rubyの文法」 + 旧「Ruby言語仕様」 →  新「Ruby言語仕様」
* 旧「組み込みライブラリ」 + 旧「標準添付ライブラリ」 → 新「クラスリファレンス」
* 旧「拡張ライブラリリファレンス」を統合
* FAQ はリファレンスマニュアルから分離
