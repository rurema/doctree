## プロジェクトのサイト

このプロジェクトで作成したリファレンスマニュアルは、
http://docs.ruby-lang.org に置いています。

## 整備するもの

このプロジェクトでは以下の二つを整備します。

  * リファレンスマニュアル本体 (ドキュメント)
  * それを表示したり変換したり検索したりするツール ([[BitClust]])

いずれも Git レポジトリに保管されています。
プロジェクトメンバーは、必ずこの二つを両方とも入手しておいてください。
レポジトリの詳しい内容や、
チェックアウト方法などは [[GitRepository]] のページに書いてあります。
