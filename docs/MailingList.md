ruby-reference-manual メーリングリストについて。

**QuickML の仕様により、長期間が投稿がなかったため消滅しました。**
以下は古い情報です。

プロジェクトに参加したい人は必ずメーリングリストに加入してください。
メーリングリストのアドレスは ruby-reference-manual ＠ ml.fdiary.net です。

## 参加方法

ruby-reference-manual メーリングリストは QuickML で運営されているので、
以下のアドレスにメールを送れば参加できます。

```
To: ruby-reference-manual ＠ ml.fdiary.net
Cc: null ＠ jp.rubyist.net
```

※ いずれのアドレスも「 ＠ 」のまわりの空白を削り、
全角「＠」を半角「@」に変換してください。

上記の登録用メールからメーリングリストに配信されるので注意してください。

また、null ＠ jp.rubyist.net は ML 登録専用のアドレスで、
誰もメールを受信しません。
