# module Logger::Severity

[lib:logger] で使用するログレベルを定義したモジュール。

## Constants
### const DEBUG -> Integer

ログレベル:デバッグを表す定数です。

### const INFO  -> Integer

ログレベル:情報を表す定数です。

### const WARN  -> Integer

ログレベル:警告を表す定数です。

### const ERROR -> Integer

ログレベル:エラーを表す定数です。

### const FATAL -> Integer

ログレベル:致命的なエラーを表す定数です。

### const UNKNOWN -> Integer

ログレベル:不明なエラーを表す定数です。
