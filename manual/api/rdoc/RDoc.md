# module RDoc

rdoc ドキュメントを扱うためのモジュールです。

## Constants

### const DOT_DOC_FILENAME -> ".document"

rdoc コマンドが処理するファイル名の一覧が書かれたファイルの名前を返します。

#@since 1.9.1

### const VERSION -> String

RDoc のバージョンを表す文字列を返します。
